# 类的使用

JavaScript 的传统方法是通过构造函数实现类的概念。而在 ES6 中，我们终于迎来了 Class 的概念。

TypeScript 除了实现了所有 ES6 中的 Class 的功能以外，还添加了一些新的用法。

这一节主要介绍 Class 的用法，下一节再介绍如何定义类的类型。
